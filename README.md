```sh
npm run test
```

## PATH 설정
### DB 연결 시 필요한 작업 
**환경변수 설정시 MONGO_DB키 값 설정**  
~~~
mongodb+srv://test:test@cluster0-xg1lk.mongodb.net/test?retryWrites=true&w=majority
~~~ 
### 필수 모듈
**nodemon global 설치**  
~~~
npm install --global nodemon
~~~
